## Reporting a Vulnerability

We take the security of our systems and the data of our users seriously. If you believe you have found a security vulnerability in any of our projects, please report it to us as soon as possible.

**Please do not report security vulnerabilities via public GitHub issues.**

Instead, please send a detailed report to:

**[security@milehigh-world.com](mailto:security@milehigh-world.com)**

### What to include in your report

To help us better understand and address the issue, please include as much information as possible:

*   **Project Name**: The name of the specific repository or project.
*   **Description**: A detailed description of the vulnerability.
*   **Steps to Reproduce**: Detailed steps to reproduce the issue (including any code or tools used).
*   **Potential Impact**: How this vulnerability could be exploited and what the potential impact is.
*   **Environment**: Details about the environment where the vulnerability was found (e.g., OS, Node version, Unity version).

### Our Commitment

We will acknowledge receipt of your report within 48 hours. We will investigate the issue and provide you with updates on our progress toward a resolution. We request that you maintain confidentiality until we have had a chance to address the vulnerability.
